# Ruby で ラーメン屋

Ruby でラーメン屋を作ってみました。

## 開発環境

- Ruby 2.6.6

## 実行方法

```
ruby main.rb
```

### 注意

`pry` が入っていない場合は次を実行して下さい。

```
gem install pry
```

もしくは，`main.rb` の `require "pry"` を削って下さい。

## 実行例

```
ようこそ ラーメン屋 たけ へ
当店のメニューはこちらです
******************************
 1. 味噌ラーメン            :  700円
 2. とんこつラーメン        :  800円
 3. 伊勢海老ラーメン        : 3200円
******************************
ラーメンを選んでください > 2
とんこつラーメン ですね。800円 になります
3000円 頂戴します
お釣りは 2200円 です
ゆっくりとんこつラーメンをお楽しみください
```

## 補足

- 存在しないメニュー番号への対処はしていません
- 代金が足りない場合の処理もしていません
- 複数行の出力はヒアドキュメント`<<~`を利用した方がよいでしょう
- 以下のように `User` クラスにインスタンス変数 `@items` あたりを用意して，購入した商品を入れる形式とした方がよいかもしれません

```rb
class User
  attr_reader :money

  def initialize(money:)
    @money = money
    @items = []
  end

  def choice(items)
    print "ラーメンを選んでください > "
    num = gets.to_i
    @items << items[num - 1]
  end
end
```
